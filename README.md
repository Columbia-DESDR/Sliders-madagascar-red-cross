# Sliders-Madagascar_red_cros
This repository hosts the code for the Madagascar red cross-specific deployment of the DESDR Sliders project.

Sliders is a web-based analytics platform designed for stakeholders including government officials, researchers, and policymakers who develop insurance programs. The included queries demonstrate the potential for complex real-world data analysis but are illustrative and may differ from those used in full operational deployment.

For more information on the original Sliders platform, visit the original repository.
